Python Fundamentals Practice

This repository contains my practice tasks based on Python Fundamentals, completed step by step from Beginner to Advanced level.

Covered Topics

Variables and Data Types

Conditional Statements (if, elif, else)

Loops (for, while)

Intermediate: Grade system, number statistics, password validation

Advanced: Scope demonstration, number counters, calculator logic
